# Rocket Avionics System

## Overview

This project focuses on the development of a complete avionics system for a competition rocket.

The system is responsible for collecting flight data, monitoring sensors, transmitting telemetry, and managing recovery mechanisms.

---

## Features

- Real-time telemetry transmission
- Dual pressure sensor support
- GPS integration
- LoRa communication
- Pyrotechnic recovery control
- Ground station compatibility

---

## Technologies

- STM32
- Modern C++
- LoRa
- GPS
- KiCad
- Git
